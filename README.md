# Universe Of Love Animation

A lightweight, responsive web animation built with HTML, CSS and JavaScript that visualizes a calming "Universe of Love" scene — perfect as a background, demo, or small interactive art piece.

## Features

- Pure HTML, CSS and JavaScript — no external dependencies
- Lightweight and easy to embed in any web page
- Responsive layout that works on desktop and mobile
- Easy to customize (colors, sizes, animation speed) via the CSS and JS files

## Live Preview

- To preview locally: open `index.html` in your browser.
- For development with hot reload: use the VS Code Live Server extension or run a static server (e.g., `npx http-server`).
- To host on GitHub Pages: enable Pages in repository settings and point it to the repository root or the `gh-pages` branch.

## Installation

Clone the repository:

```bash
git clone https://github.com/BinaryVortex/Universe-Of-Love-Animation.git
cd Universe-Of-Love-Animation
```

Then open `index.html` in your browser, or serve the folder with a simple static server.

## Usage & Customization

- Styling: edit `styles.css` (or the relevant CSS file) to change colors, background, and layout.
- Animation/behavior: edit `script.js` (or the relevant JS file) to adjust speeds, particle counts, or interaction handlers.
- Replace the screenshot used in this README with a high-quality capture of the animation:
  - Save the image as `screenshot.png` in the repo root (or update the path in the README).
  - Commit the image so the README preview shows the screenshot on GitHub.

Example quick-tweaks:
- Change primary colors in CSS variables (if present) or directly in the stylesheet.
- Reduce animation CPU by lowering particle counts or disabling shadow/blur effects.
